# StucareAssignment


This project demostrates use of TMDB movie API implementation using BLoC architecture pattern and will help to understand use of Streams in flutter.

Following components are used in this project:

- [MVVM]
- [DataBinding]
- [LiveData]
- [Retrifit]
- [Glid]
